
## Principles

**Merge Sort**
* Complexity
  * T: O(N*logN)
  * S: O(N)
* Key Points
  * Can not swap on the original array. 

**Heap Sort**
  * Complexity
    * T: O(N*logN)
    * S: 
      * O(logN) Recursive
      * O(1) iterative
  * Key Points
    * In-place sorting
    * not stable. May make stable

**Problems**
* LC912 sort an array
  * Merge Sort. Space O(N)
